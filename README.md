Hey there. I'm Jay Kroger, a Platform Security professional. 

## Specialty

A close mentor of mine told me early in my career that you can't stop a truly motivated adversary. You have to frustrate them, make them give up, make them go somewhere else. 

I'm passionate about security architecture that makes adversaries think twice. I spend a significant amount of time researching secure platform architecture patterns across different domains, including cloud security, network security, secrets management, and logging & detection. Coming from backgrounds in both software development and cybersecurity, I understand the need to balance accelerated delivery with security and governance.
